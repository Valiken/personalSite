My name is Brian Berg

I am a recent graduate from Cal Poly Pomona with a BS in Business Administrator and a concentration in Computer Information Systems. I am currently working for CGI as a Programmer Analyst, specalizing in Configuration Management.

My Projects

Finger Tip Facts Mobile App

For my senior project my team and I built a hybrid mobile application using jQuery, HTML, and CSS that used a JSON API to retreive and read data from the school districts website.

Snack Tzar Application

Built a simple application utilizing AngularJS and Firebase to trace snack requests for the client site I am working on at CGI. Made restocking the snack cube at work easier to coordinate.

ITC Competiton

Competed in ITC Competiton at Cal Poly Pomona with a team of five people. Placed first in the computer programming section. Built with VB.net in roughly 2 weeks worth of time during Spring 2014

Inventory App POF

Helped develop a proof of concept for a barcode scanning app that would help augment CGI's ERP Inventory Module. Built the proof of concept using Twitter Bootstrap, AngularJS and PhoneGap
